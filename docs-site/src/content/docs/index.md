LiveCharts is a unified chart abstraction layer for the Laravel framework. It eliminates the friction of integrating JavaScript charting libraries by providing a pure PHP API to define, configure, and render charts.

## Features

- **Pure PHP API**: Define everything in PHP you already know.
- **Multi-Engine**: Switch between ApexCharts and Chart.js with a single method call.
- **Livewire Native**: Seamless integration with Livewire v3/v4 lifecycles.
- **Reactive**: Charts react automatically to Livewire property updates.
- **Artisan Ready**: Scaffold chart classes with `php artisan make:chart`.
- **Zero JS Required**: No custom JavaScript needed for standard use cases.
- **Customizable**: Full access to engine-native options when you need them.

## Supported Engines

| Engine | Version |
|---|---|
| ApexCharts | ^5.10.6 |
| Chart.js | ^4.5.1 |
